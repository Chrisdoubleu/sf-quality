# Evidence and Confidence Rules

- Every critical finding row must include `EvidencePath` (file + sheet/page/image reference).
- If a mapping is not exact, mark it `INFERRED` and explain the inference.
- If data is unavailable, use `UNKNOWN` (do not invent values).
- Confidence values: `High`, `Medium`, `Low`.
- Keep quotes from forms concise; prioritize structured tables over narrative.
